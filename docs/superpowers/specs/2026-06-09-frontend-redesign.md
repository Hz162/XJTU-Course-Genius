# XJTU Course Genius 前端重设计

## 1. 导航架构

当前问题：8 个功能按钮挤在 AppBar actions 中，桌面端体验差，无层级分组。

### 方案：侧边栏抽屉

```
┌──────────────┬──────────────────────────┐
│ 📚 课程浏览    │                          │
│  ├ 已选课程    │   当前视图内容              │
│  ├ 主修推荐(TJKC)│                          │
│  ├ 方案内(FANKC)│                          │
│  ├ 方案外(FAWKC)│                          │
│  ├ 基础通识(XGXK)│                          │
│  └ 体育(TYKC)  │                          │
│               │                          │
│ 🎯 操作        │                          │
│  ├ 选课控制    │                          │
│  └ 配置管理    │                          │
│               │                          │
│ 📍 校区: [兴庆▼]│                          │
└──────────────┴──────────────────────────┘
```

- 侧边栏宽度 ~200px，可折叠
- 选中项高亮（蓝底白字）
- 校区选择器常驻侧边栏底部
- 课程类型切换不丢失搜索/选课状态

## 2. 视觉风格

- 主色 `#409EFF`，浅色背景 `#F5F7FA`，保持 Element Plus 蓝白风格
- Material 3 组件体系 (`useMaterial3: true`)
- 卡片圆角 12px，按钮/输入框圆角 8px
- 卡片阴影提升为 `boxShadow: 0 4px 24px rgba(0,0,0,0.06)`
- 主按钮使用渐变 `linear-gradient(135deg, #409EFF, #66B1FF)`
- Logo 可更换：读取 `assets/logo.png`，缺失时用文字回退

## 3. 字体

- 打包 **Noto Sans SC** (~8MB)，确保 CJK 跨平台渲染一致
- `pubspec.yaml` 声明 font family，`appTheme()` 中设为默认
- 解决 Windows 上 Flutter CJK 字体回退导致的粗细不均

## 4. 页面设计

### 4.1 登录页

- 居中卡片 (400×auto, 圆角 16px, 投影)
- 顶部：Logo 图片 (assets/logo.png) / 文字回退
- 文字："西安交通大学 · 统一身份认证"
- 输入框：账号/密码整合在一个白色卡片内，带图标
- 验证码：按需展开，可点击刷新
- 按钮：渐变蓝底 "登录"，底部灰色提示 "Ctrl+Enter 快速登录"
- 加载状态：按钮内转圈替代文字

### 4.2 MFA 页

- 步骤引导：编号圆圈 (1 → 2)，引导用户
- 方式选择：两张并排卡片（手机短信/邮箱），选中态蓝边框+淡蓝背景
- 初始化自动触发，无需手点"初始化验证"
- 目标号码/邮箱清晰显示
- 输入框 + 倒计时按钮合一横排
- 验证按钮：渐变蓝底

### 4.3 轮次选择页

- 保持简单：居中 dropdown + 确认按钮
- 新增：轮次描述文字（时间范围、状态），帮助判断选哪个

### 4.4 主页（侧边栏 + 内容区）

**课程浏览面板**（默认视图）：
- 搜索框 + 按钮横排，每行数据末尾 "+抢课" 按钮
- 替代当前的 checkbox + 保存两步操作
- 数据用卡片列表替代 DataTable（避免横向滚动）
- 已选课程视图额外显示 ✓/× 结果

**选课控制面板**：
- 独立视图，侧边栏点击进入
- 大面积开始/停止按钮（红/绿色，圆形 icon）
- 进度条 + "进度 N/M · 已抢到 K"
- 实时日志区域（滚动文本框）

**配置管理面板**：
- 抢课列表：可滚动卡片，每项显示课程编号+名称+冲突数量
- 展开/弹窗管理冲突课程
- 删除按钮（红色 ✕）
- 底部 "+ 添加课程" / "+ 从浏览添加"

## 5. 快捷键

| 快捷键 | 功能 | 适用页面 |
|--------|------|----------|
| Ctrl+Enter | 登录 / 提交 | 登录页, MFA页 |
| Ctrl+S | 保存配置 | 配置页 |
| F5 | 刷新课程 | 课程浏览 |
| Ctrl+B | 开始/停止抢课 | 全局 |
| Ctrl+1~6 | 切换课程类型 | 课程浏览 |
