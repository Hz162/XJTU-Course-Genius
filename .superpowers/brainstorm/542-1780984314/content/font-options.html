中文字体方案

当前 Flutter 默认字体在 Windows 上 CJK 渲染不稳定，部分字粗细不一。选择一个方案。

A — 系统字体 (Microsoft YaHei)

指定系统自带微软雅黑，零体积增加。Windows 预装，兼容性好。用 fontWeight 控制粗细。

B — 打包 Noto Sans SC

Google 开源字体，跨平台完全一致。需将 .ttf 打包进 app，增加约 8MB。macOS/Linux 也能用。

C — 智能回退链

指定一个字体家族链：PingFang SC → Microsoft YaHei → Noto Sans CJK。各平台自动取第一个可用的，零体积。
