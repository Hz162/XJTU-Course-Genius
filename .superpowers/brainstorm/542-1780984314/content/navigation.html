导航布局选哪个？

当前所有功能挤在 AppBar 的一排 TextButton 里，桌面端很拥挤。哪种导航模式更适合?

A

左侧 NavigationRail

Material 3 标准方案，左侧竖排图标 + 标签，选中态高亮。功能分组：课程浏览、选课控制、配置管理。桌面端原生感强，空间利用率高。

B

底部 Tab Bar

类似移动端 BottomNavigationBar，顶部放标题 + 状态，底部切Tab：已选 / 查询 / 抢课 / 配置。直观但纵向空间被压缩。

C

侧边抽屉 + 顶部面包屑

左侧可折叠抽屉放功能入口，大面积留给内容。列表类桌面应用的经典布局（如 VS Code 侧栏）。

D

保留顶部 Tab Bar

在 AppBar 下方用 TabBar 切换视图，替代挤在 AppBar actions 里的一排按钮。改动最小，但纵向空间仍然有限。
